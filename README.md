# Hospital Readmission Prediction


## Overview

This project aims to built an end-to-end data science pipeline for predicting hospital readmission using the Diabetes 130-US Hospitals dataset.

This repository documents my learning journey as I study data science while applying every concept to a real-world project.

The project will gradually evolve form:

- Data Understanding
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Feature Selection
- Machine Learning
- Explainable AI
- Deployment
