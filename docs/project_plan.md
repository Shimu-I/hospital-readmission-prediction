# Hospital Readmission Prediction

## Project Goal

Build and end-to-end data science project that predicts whether a diabetic patient will be readmitted to a hospital after discharge.

The goal is not only to build a machine learning model but also learn every stage of the data science lifecycle.

---

## Problem Statement

Hospital readmissions are costly for both hospitals and patients.

If patients at high risk of readmission can be identified early, healthcare providers may be able to improve treatment plans and reduce unnecessary readmissions.
